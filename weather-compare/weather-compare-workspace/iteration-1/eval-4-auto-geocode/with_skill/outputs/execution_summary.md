## 技术细节

### 地理编码API
- 使用Open-Meteo Geocoding API
- URL: `https://geocoding-api.open-meteo.com/v1/search`
- 参数: `name`, `count`, `language=zh`, `format=json`

### 天气数据API
- 使用Open-Meteo Forecast API
- URL: `https://api.open-meteo.com/v1/forecast`
- 参数: `latitude`, `longitude`, `daily`, `timezone=Asia/Shanghai`, `forecast_days=16`, `models=ecmwf_ifs,gfs_seamless`

### 数据模型
- ECMWF IFS（欧洲中期天气预报中心）
- GFS Seamless（美国全球预报系统）
