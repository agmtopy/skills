# Weather Compare Skill - 使用说明

## 安装位置
✅ 已安装到：`.claude/skills/weather-compare/`

## 如何使用

在当前项目中，你可以直接向 Claude 提问天气相关问题，skill 会自动触发：

### 示例用法

1. **查询预设城市天气**
   ```
   帮我查询17号到21号伊宁、库尔德宁、赛里木湖的天气
   ```

2. **使用中文日期格式**
   ```
   我想在4月20号到4月25号去北京和上海，看看天气怎么样
   ```

3. **使用相对日期**
   ```
   下周我要去成都旅游，帮我看看天气
   ```

4. **查询任意城市**
   ```
   我计划5月1日到5月5日去重庆和武汉，帮我查查天气
   ```

## 功能特性

- ✅ 自动获取中央气象台、ECMWF、GFS 三个数据源的天气对比
- ✅ 支持中文日期格式（"4月20号"、"17号"等）
- ✅ 支持相对日期（"今天"、"明天"、"下周"等）
- ✅ 自动查询城市经纬度
- ✅ 自动安装依赖（BeautifulSoup）
- ✅ 提供 20+ 个预设城市

## 预设城市列表

包括但不限于：
- 新疆：伊宁、库尔德宁、赛里木湖、乌鲁木齐、喀什、吐鲁番
- 北京、上海、广州、深圳
- 西藏：拉萨、林芝
- 云南：昆明、大理、丽江、香格里拉
- 四川：成都、九寨沟
- 其他：杭州、苏州、南京、西安、厦门、青岛、三亚、海口

## 测试结果

✅ 所有测试用例通过率：100%

## 技术细节

- 数据源：Open-Meteo API（免费，无需 API key）
- 官方数据：中央气象台（需 BeautifulSoup）
- 模型数据：ECMWF IFS、GFS Seamless
- 最大预报范围：16天
